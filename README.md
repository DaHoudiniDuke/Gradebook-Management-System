# Gradebook Management System

A command-line Java application that stores student records and calculates grade statistics using object-oriented programming principles.

---

## Description

This project is a gradebook system built in Java that allows users to store student information and calculate class statistics such as averages, medians, minimum scores, and maximum scores.

The program demonstrates core object-oriented programming concepts, including encapsulation, class interaction, and data organization using ArrayLists.

---

## Features

- Store student information (first name, last name, PID, score)
- Automatically convert numeric scores to letter grades
- Calculate class statistics:
  - Average score
  - Median score
  - Minimum score
  - Maximum score
- Retrieve student names and grades by PID
- Command-line interaction for user input

---

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- IntelliJ IDEA

---

## Project Structure

Grade.java – Handles numeric scores and converts them to letter grades.

Student.java – Stores student information such as first name, last name, PID, and grade.

Gradebook.java – Manages the collection of students and performs calculations such as average, median, minimum, and maximum scores.

Main.java – Runs the program and processes user commands through the command-line interface.

## How to Run

1. Clone the repository:
2. Open the project in IntelliJ IDEA or another Java IDE.
3. Run the `Main.java` file to start the program.
