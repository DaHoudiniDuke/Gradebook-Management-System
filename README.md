# Gradebook Management System

A Java-based, object-oriented gradebook application that allows users to manage students and calculate class statistics such as averages, medians, minimums, and maximums.

---

## Features
- Add students with first name, last name, PID, and score
- Automatically converts numeric scores to letter grades
- Calculate:
  - Average score
  - Median score
  - Minimum and maximum scores
- Display all students in a formatted table
- Command-line interface (CLI)

---

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- IntelliJ IDEA
